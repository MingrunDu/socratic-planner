# Manifest

This repository packages seven Socratic Codex skills.

| Path | Purpose |
| --- | --- |
| `dist/codex/skills/socratic-planner/` | Vague idea to approved executable plan |
| `dist/codex/skills/socratic-prompt-midwife/` | Vague prompt or approved plan to bounded execution prompt |
| `dist/codex/skills/socratic-plan-midwife/` | Combined plan + prompt midwife workflow |
| `dist/codex/skills/socratic-knowledge-teacher/` | Beginner-friendly Socratic teaching |
| `dist/codex/skills/socratic-paper-reading-agent/` | Evidence-anchored paper reading and reproducibility analysis |
| `dist/codex/skills/socratic-paper-finder/` | Literature search and evidence table workflow |
| `dist/codex/skills/socratic-method-cooking/` | Empirical research gap and methodology-flow matching |
| `dist/markdown/` | Single-file Markdown exports of the same seven skills |
